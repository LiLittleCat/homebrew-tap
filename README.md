# homebrew-tap

Homebrew tap for [LiLittleCat](https://github.com/LiLittleCat)'s projects.

## Usage

### Add the tap

```bash
brew tap LiLittleCat/tap
```

### Install fonts

```bash
# Simplified Chinese variant
brew install --cask font-cascadia-next-sc-nerd-font

# Traditional Chinese variant
brew install --cask font-cascadia-next-tc-nerd-font

# Japanese variant
brew install --cask font-cascadia-next-jp-nerd-font
```

### Uninstall

```bash
brew uninstall --cask font-cascadia-next-sc-nerd-font
```

## Available Casks

| Cask | Description |
|------|-------------|
| `font-cascadia-next-sc-nerd-font` | Cascadia Next SC Nerd Font — Simplified Chinese |
| `font-cascadia-next-tc-nerd-font` | Cascadia Next TC Nerd Font — Traditional Chinese |
| `font-cascadia-next-jp-nerd-font` | Cascadia Next JP Nerd Font — Japanese |

> Fonts include 7 weights: ExtraLight, Light, Regular, Medium, SemiBold, Bold, ExtraBold.

## Source

Font files are built from [Cascadia-Next-Nerd-Font](https://github.com/LiLittleCat/Cascadia-Next-Nerd-Font).
